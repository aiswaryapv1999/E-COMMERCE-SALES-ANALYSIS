# E-COMMERCE-SALES-ANALYSIS

**Project Overview** 

This project leverages Power BI to analyze e-commerce sales performance and provide actionable insights into targets, profitability, customer behavior, and geographic trends. The dashboard integrates multiple visualizations and calculated measures to deliver a comprehensive view of business performance.


**KEY ANALYSES & VISUALS**

- Sales Target Achievement by Category (Bar Chart): Compare actual vs. target sales across product categories.
- Total Sales vs. Target (Gauge): Quick snapshot of overall target achievement.
- Calculated Measures:
    * Count of Orders
    * Year-to-Date (YTD) Sales using a Date Table
    * Average Profit in Delhi
    * Maximum Profit Margin by Sub-Category (Pie Chart)
    * Minimum Target Values by Category (Multi-Row Card)
- Monthly Sales Trend (Line Chart): Track seasonality and growth patterns.
- Profit vs. Quantity by Sub-Category (Scatter Chart): Identify relationships between sales volume and profitability.
- Sales Distribution by Sub-Category (Tree Map): Visualize contribution of each sub-category to total sales.
- Sales Performance Matrix: Tabular view of KPIs across categories and regions.
- Geographic Sales Analysis (Map): Regional breakdown of sales performance.
- Order Count Analysis by State (Funnel Chart): Rank states by order volume.

**Tools & Techniques**

- Power BI Desktop – Data modeling, DAX measures, and interactive dashboards
- DAX Calculations – YTD sales, profit margins, and custom KPIs
- Data Cleaning & Transformation – Using Power Query for structured analysis

**Impact**

This dashboard empowers stakeholders to:
- Monitor sales target achievement at multiple levels
- Identify high-performing categories and regions
- Optimize inventory and marketing strategies based on trends
- Make data-driven decisions with interactive, drill-down insights
